Instantiates a const QueryTerm instance:

  • term is the term that will be looked up in the index;
  • termPosition is the zero-based position of the term in an ordered list of all the terms in the source text;
  • modifier is the modifier used for this term;
  • index is the zero-based position of the start of the first character of term in the source text string (deprecated); and
  • position is the position of the term from the start of the source text as a fraction of the source text length (deprecated).
